\# Cancer Detection Using Machine Learning



\## Project Overview

This project builds a machine learning model to classify breast cancer tumors as malignant or benign using the Breast Cancer Wisconsin dataset.



\## Tools Used

Python

Pandas

NumPy

Matplotlib

Seaborn

Scikit-learn



\## Dataset

\- 569 samples

\- 30 tumor features

\- Target labels:

&nbsp; - 0 = Malignant

&nbsp; - 1 = Benign



\## Workflow

1\. Load dataset

2\. Convert dataset to dataframe

3\. Explore data

4\. Split into training and testing sets

5\. Train Random Forest model

6\. Evaluate model accuracy

7\. Visualize results



\## Results

Model accuracy: 96%
